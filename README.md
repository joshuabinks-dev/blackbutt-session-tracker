# Blackbutt Session Tracker (Vite + React) — v0.1

Fresh rebuild from the v1.0 technical specification.

## Run locally
1. Install Node.js LTS (18+ recommended)
2. In this folder:

```bash
npm install
npm run dev
```

Open the local URL (usually http://localhost:5173)

## Quick test
1. Home → Templates → Start
2. Session → Group A / B → Start rep
3. Tap athletes as they finish (manual blocks)
4. Results → Copy TSV / Share image

## GitHub Pages (later)
Edit `vite.config.ts` and set `base: "/<repo-name>/"`, then run `npm run build` and deploy the `dist/` folder.
